# blackout

## How to run development server (deprecated)
1. Clone repository
```
git clone https://github.com/WebUraXalys/blackout.git
```

2. Switch directory
```
cd blackout
```

3. Run with docker-compose
```
docker compose up
```

4. Appliaction will appear on localhost:80
